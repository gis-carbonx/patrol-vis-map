# Digital Map Patrol Monitoring

Interactive digital map for patrol monitoring, camera trap management, forest disturbance alerts, and fire event monitoring.

<img width="1919" height="949" alt="image" src="https://github.com/user-attachments/assets/a2848d70-e8b8-49c2-8803-62588af71db6" />

## Overview

This application integrates field patrol data, camera trap observations, forest disturbance alerts, and fire detections into a single digital map platform.

The system supports the implementation of the Sanggala Corridor Project.

---

## Main Features

### Patrol Monitoring

- Patrol point visualization
- Monthly Patrol track visualization

### Camera Trap Monitoring

- Camera trap location mapping
- Detection intensity grid

### Disturbance Monitoring

- Global Forest Watch (GFW) alert integration

### Fire Monitoring

- NASA FIRMS hotspot integration
